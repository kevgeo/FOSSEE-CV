findCirclesGrid

Finds centers in the grid of circles

Calling Sequence

[centers blob ]= findCirclesGrid(Image,circ_row,circ_cols,flag);

Parameters

Image :

Path of image file

double circ_row,circ_cols :

Number of circles per row and column

string flags :

Different methods to find centers

1.CALIB_CB_SYMMETRIC_GRID :

uses symmetric pattern of circles

centers :

array of detected corners

blob :

feature detector that find blobs

Description

This function finds centers in the grid of circles

Examples

inImage = imread("chessboard3.png");
[centers blob ]= findCirclesGrid(inImage,7,7,"CALIB CB SYMMETRIC GRID");
