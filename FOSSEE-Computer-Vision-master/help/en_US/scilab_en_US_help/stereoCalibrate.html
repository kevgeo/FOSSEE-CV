stereoCalibrate

Takes in 3D points and outputs the corresponding 2D points in image plane

Calling Sequence

[rotMatrix Transvec EMatrix FMatrix] = stereoCalibrate(list_size,calibPoints,ProjcalibPoints1,ProjcalibPoints2,camera1,dist1,camera2,dist2,width,height);

Parameters

matrixofdouble calibPoints :

vector of vector of calibration points

matrixofdouble ProjcalibPoints1 :

vector of vector of projection of calibration points by first camera

matrixofdouble ProjcalibPoints2 :

vector of vector of projection of calibration points by second camera

matrixofdouble camera1 :

first camera matrix

matrixofdouble dist1 :

vector of distortion coefficients for first camera

matrixofdouble camera2 :

second camera matrix

matrixofdouble dist2 :

vector of distortion coefficients for second camera

double width,height :

size of image in pixels

rotMatrix :

rotation matrix

TransVec :

translation vector

EMatrix :

essential matrix

FMatrix :

fundamental matrix

Description

The function outputs the corresponding 2D points in image plane.
