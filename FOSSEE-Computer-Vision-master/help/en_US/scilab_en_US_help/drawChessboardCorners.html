drawChessboardCorners

Displays the detected chessboard corners

Calling Sequence

img = drawChessboardCorners(Image,pts_row,pts_cols,coordinates,found);

Parameters

Image :

Path of image file

double pts_row,pts_cols :

Number of inner corners per chessboard per row and column

matrixofdouble coordinates :

array of detected corners

found :

boolean value indicating whether complete board has been found or not

img :

Image with deteced chessboard corners

Description

This function displays the detected chessboard corners.

Examples

inImage = imread("chessboard3.png");
[found coordinates ] = findChessboardCorners(inImage,7,7,"CV_CALIB_CB_ADAPTIVE_THRESH");
img = drawChessboardCorners(inImage,7,7,coordinates,found);
