showMatchedFeatures

Displays the matched features between two images

Calling Sequence

outImage = showMatchedFeatures(Image1,Image2);

Parameters

Image1 :

First image file path

Image2 :

Second image file path

outImage :

Output image displaying the matched features

Description

The function takes as input two images and returns an image displaying the matched features by drawing lines from one matched feature to the other.

Examples

Reading first image
I1 = imread("left.jpg");
//Reading second image
I2 = imread("right.jpg");
//Executing function
outImg = showMatchedFeatures(I1,I2);
