epipolarlines

Draws epipolar lines coressponding to the two images

Calling Sequence

[out1 out2] = epipolarlines(I1,I2)

Parameters

I1 :

First input image

I2 :

Second input image

out1 :

Epipolar lines in first image

out2 :

Epipolar lines in second image

Description

The function takes in two images and outputs two images displaying the epipolar lines in each one.

Examples

I1 = imread("left.jpg");
I2 = imread("right.jpg");
[I3 I4] = epipolarlines(I1,I2);
