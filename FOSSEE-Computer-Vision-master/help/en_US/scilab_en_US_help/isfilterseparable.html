isfilterseparable

Checks whether the given filter/kernel is separable or not.

Calling Sequence

bool check = isfilterseparable("filter",kernel)

Parameters

filter:

filter/kernel values

check :

Boolean variable which tells whether the kernel is separable(1) or not(0).

Description

The function finds the rank of the kernel. If it is 1, then the function returns a boolean value 1 signifying that the filter is separable; else it returns boolean value 0 signifying it is not separable.

Examples

//Sobel filter
sobel = [-1 0 1; -2 0 2; -1 0 1];
//Executing function
isfilterseparable("filter",sobel)
